# DozePlus v1.1.0 🇺🇸
- Added Global Task Killer: action.sh now scans and kills background apps not in Recents
- Added SePolicy Injection: prevents Android 16 from blocking module actions
- Added Verification System: reports if a process fails to terminate
- Process Protection: current foreground app is never closed
- Code Refactor: removed unnecessary comments for better performance
